### [백준 1835 : 카드 ](https://www.acmicpc.net/problem/1835)

#### 제출 (1트 성공)
```python
n,m = map(int, input().strip().split())

r_word = {0 : "zero", 1 : "one", 2 : "two", 3 : "three", 4 : "four", 5 : "five", 6 : "six", 7 : "seven", 8 : "eight", 9 :"nine"}

def trans_n2w(n) :
    n10 = n // 10
    n1 = n - (10 * n10)
    if n10 == 0 :
        return r_word[n1]
    return f"{r_word[n10]} {r_word[n1]}"

result = sorted(list(range(n, m+1)), key=lambda x : trans_n2w(x))

for i in range(len(result)) : 
    print(result[i], end=" ")
    if i != (len(result) - 1) and (i % 10) == 9 :
        print()

# 찾아본 것들
# sort에 정렬기준으로 람다식 전달하는 것
# print에 end 인자 사용하는 것
```

#### 참고답안
```python
def number_to_words(num):
    num_to_word = ["zero", "one", "two", "three", "four", "five", "six", "seven", "eight", "nine"]
    return " ".join(num_to_word[int(digit)] for digit in str(num))

def main():
    import sys
    input = sys.stdin.readline
    M, N = map(int, input().split())
    numbers = [(i, number_to_words(i)) for i in range(M, N + 1)]
    numbers.sort(key=lambda x: x[1])
    sorted_numbers = [num[0] for num in numbers]

    for i in range(0, len(sorted_numbers), 10):
        print(" ".join(map(str, sorted_numbers[i:i + 10])))

if __name__ == "__main__":
    main()
```
> `number_to_words` 구현
1. **string도 iterable이라서...**
굳이 map까지 만들지 않았다.
str로 문자열로 컨버팅하고, 문자열의 각각의 char에 대해서 다시 int로 컨버팅해서 list에서 num에 해당하는 word를 찾도록 함. 코드가 매우 간결해짐.
2. **map이 아니라 튜플리스트사용**
`map(dict)`는 키를 기준으로 정렬하려면 추가 작업이 필요하고, 정렬 후 순설르 유지하지 않을 수 있음. 튜플리스트가 정렬 구현하기 훨씬 용이함.
3. **출력에서 range 설정**
하나의 라인에 10개의 요소만 출력하기 위해서 `range(0, len, 10)`으로 10씩 증가하게 하고, loop 문에서는 인덱스 범위를 활용해서 슬라이싱해서 사용.
