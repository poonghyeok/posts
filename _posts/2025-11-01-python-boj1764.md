### [백준 1764 : 듣보잡](https://www.acmicpc.net/problem/1764)

#### 제출 (시간초과)
```python
import sys
input = sys.stdin.readline

def name2uninum(nmstr) : 
    return int("".join([str(ord(c)) for c in nmstr]))

db_nums = input().split(" ")
dnum, bnum = int(db_nums[0]), int(db_nums[1])
d_words = []
b_words = []

for _ in range(dnum) :
    d_in = input()
    d_words.append(name2uninum(d_in))
    
for _ in range(bnum) :
    b_in = input()
    if (name2uninum(b_in) in d_words) :
        b_words.append(b_in)

print(len(b_words))
for i in range(len(b_words)) :
    print(b_words[i])
```
> list와 in 사용은 문제가 없다고 생각하고 숫자로 변환해서 in 비교를 사용한다면 시간초과가 되지 않을 것이라고 계속 도전함.
문제는 자료 구조 사용에 있었음. 시간 복잡도가 가장 낮은 set을 사용해어야 했다.

#### 모범답안
```python
n, m = map(int, input().split())

no_see = set(input() for _ in range(n))
no_hear = set(input() for _ in range(m))

result = sorted(no_see & no_hear)

print(len(result))
for name in result :
    print(name)
```
1. **구조분해할당(?)**
초기값을 입력받기 위해서 `map(int, input().split())`을 사용했다.
기존에는 `input().split(" ")[0]` `input().split(" ")[1]`과 같이 사용을 했었다.
2. **set 초기화**
`set(input() for _ in range(n))`를 통해 여러 입력값을 set으로 초기화 시켰다.
`{1,2,3}`으로도 set을 초기화 할 수 있다.
3. **set intersect**
`no_see & no_hear`에서 `&`로 교집합을 구했다.
추가로 마찬가지로 `-` 로 `Aset - Bset`을 구할 수 있고
`^`로 `(A U B) - (A n B)`를 구할 수 있다.
4. **sorted**
`.sort()`는 그냥 정렬만 하고 `sorted()`는 정렬된 객체를 반환한다.

#### 응용문제로 [백준1822](https://www.acmicpc.net/problem/1822) 문제를 풀어보겠다.

**제출답안(정답)**
```python
n, m = map(int, input().split())

a_set = set(input().split())
b_set = set(input().split())

res = [int(n) for n in (a_set - b_set)]
res.sort()
print(len(res))
if len(res) > 0 :
    print(" ".join([str(n) for n in res]).strip())
```

**모범답안**
```python
n, m = map(int, input().split())  # A, B 집합 원소 개수 입력
A = set(map(int, input().split()))
B = set(map(int, input().split()))

result = sorted(A - B)  # 차집합: A에 있지만 B에는 없는 원소

print(len(result))  # 차집합 원소 개수 출력
for num in result:
    print(num)
```
> `map`으로 먼저 `int`로 변환해서 받고 `set`을 초기화하는 결정!
